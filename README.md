# Tagg

Back in 2015, I was relying on people who might have had similar thoughts about building an actual browser that **helps** us — not one that controls us. This app is dedicated to all the people who helped me build it and to the ones that are no longer here. Let's change for the better of all of us.

---

## About

**Tagg** is an open-source browser project focused on putting the user first. The vision is a browser that respects your autonomy, privacy, and freedom — designed by the community, for the community.
